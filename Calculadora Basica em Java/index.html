Calculadora Básica em Java

Projeto escolar com operações matemáticas básicas, fatorial, soma de intervalos e histórico de operações.

Funcionalidades

  • Somar
  • Subtrair
  • Multiplicar
  • Dividir (com validação de divisão por zero)
  • Fatorial
  • Soma dos intervalos
  • Histórico de operações

Como executar

No terminal, dentro da pasta do arquivo Java:

javac Calculadora.java
java Calculadora
